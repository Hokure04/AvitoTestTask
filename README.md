# Avito

## О проекте

Данный проект содержит автоматизированные тесты для API-сервиса объявлений, предоставленного в рамках тестового задания. Проект выполнен с использованием Java 17.  

В рамках проекта проверяются основные сценарии работы API:

- создание объявления
- получение объявления по идентификатору
- получение всех объявлений продавца
- получение статистики по идентификатору объявления
- удаление объявления

Тесты покрывают:

- позитивные сценарии
- негативные сценарии
- граничные значения
- E2E-сценарии, включающие несколько последовательных шагов
- проверки, выявляющие расхождения между ожидаемым поведением и фактической реализацией API

Если API ведёт себя не в соответствии с ожидаемым результатом, тест падает. Такие случаи дополнительно описаны в файле `BUGS.md`.


## Структура проекта

- `src/main/java` — DTO, API client, утилиты для чтения тестовых данных и генерации sellerId
- `src/main/resources/TestData` — JSON-файлы с тестовыми данными
- `src/test/java` — автоматизированные тесты
- `TESTCASES.md` — описание тест-кейсов
- `BUGS.md` — найденные дефекты
- `README.md` — инструкция по запуску проекта

## Запуск проекта
Для полной сборки проекта необходимо выполнить команду:  
`mvn clean install`

Для запуска всех тестов выполнить команду:
`mvn clean test`

Для запуска отдельного тест-класса:
`mvn -Dtest=TestClassname test`

Для запуска отдельного метода в тестовом классе:
`mvn -Dtest=TestClassname#testMethodName test`

Для генерации отчётов с помощью allure:
`mvn allure:report`
